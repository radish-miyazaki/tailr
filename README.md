## tailr

```
Rust tail

Usage: tailr [OPTIONS] <FILE>...

Arguments:
  <FILE>...  Input file(s)

Options:
  -c, --bytes <BYTES>  Number of bytes
  -n, --lines <LINES>  Number of lines [default: 10]
  -q, --quiet          Suppress headers
  -h, --help           Print help
  -V, --version        Print version
```
